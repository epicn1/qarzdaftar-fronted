# Qarz Daftari - Frontend

Qarz Daftari uchun frontend ilova (HTML, CSS, JavaScript).

## Features

- 📱 Responsive design
- 🎨 Modern UI
- 🔍 Search functionality
- 💰 Real-time calculation
- 🔗 REST API integration

## Files

- `index.html` - HTML markup
- `index.js` - JavaScript logic
- `style.css` - Styling

## Setup

1. Ensure backend is running on `http://localhost:5000`
2. Open `index.html` in browser or serve with backend

## API Integration

Frontend connects to backend API endpoints:
- `GET /api/debts` - Fetch all debts
- `POST /api/debts` - Add new debt
- `DELETE /api/debts/<id>` - Delete debt

## Technology

- HTML5
- CSS3 (Responsive)
- Vanilla JavaScript
- Fetch API
